  - **Active Tasks**: Tasks currently in progress.  
  - **Completed Tasks**: Successfully finished tasks.  
  - **Failed Tasks**: Tasks marked as failed.  
- Interactive cards to:  
  - Accept new tasks.  
  - Mark tasks as completed or failed.
